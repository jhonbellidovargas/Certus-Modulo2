Definiciones de Tecnologias del internet

  • HTTP

    Es un protocolo de comunicación que permite la transferencia de información entre un cliente y un servidor.

  • HTML

    HTML es un lenguaje de marcado que se utiliza para el desarrollo de páginas web. Es el estándar para la creación de páginas web y se utiliza para crear y estructurar diferentes páginas de un sitio web.

  • URL

    Es un identificador de recursos en Internet. Es una dirección única que identifica un recurso en Internet.

  • TCP/IP

    Es un conjunto de protocolos que permite la comunicación entre diferentes dispositivos conectados a una red.

  • Internet

    Es una red de redes que permite la comunicación entre diferentes dispositivos conectados a una red.

  • W3C

    Es una organización internacional que se encarga de desarrollar estándares para la web.

  • Mozilla Foundation

    Es una organización sin fines de lucro que se encarga de desarrollar el navegador web Mozilla Firefox.
